# Pake online installer

Pake uses the open-source [Qt Installer Framework](https://doc.qt.io/qtinstallerframework/)
as the unattended online download and verification backend for Windows, macOS,
and Linux.

The workflow creates one QtIFW package repository per online configuration.
`repogen` stores the application payload as a platform-independent 7z component
archive. `binarycreator` produces the backend embedded by the platform carrier.
The carrier uses the same app name, icon, and visible package layout as the
offline build, so no second QtIFW wizard is displayed.

The installer definition is generated by `scripts/pake-online/qtifw.mjs`. The
generated controller checks Cloudflare's country trace before repository access
and selects the configured GitHub acceleration URL only for mainland China.

Qt Installer Framework 4.8.1 is downloaded from the official Qt release server
and verified against the pinned SHA-256 digest for each platform.
